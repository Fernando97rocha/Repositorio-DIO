

# PRINCIPAIS COMANDOS QUE APRENDI NO GIT:

git init: Cria um novo repositório local.

git ls: lista todos os diretórios e arquivos de um repositório.

git status: mostra a situação atual dos arquivos/diretórios; se estão pendentes.

git add: "Deixa o arquivo/diretório pronto para ser commitado".

git add . ou git add -a: adiciona todos os arquivos/diretórios em "espera" no git status.

git commit (sempre com comentário -m):  

git push origin main: "Empurra" a branch para o repositório remoto.
